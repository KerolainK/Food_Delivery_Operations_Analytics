# 📊 Food Delivery Operations & Customer Analytics

## 📖 Descripción del proyecto

Este proyecto consiste en un análisis exploratorio de datos (EDA) y el desarrollo de un dashboard interactivo en Microsoft Excel sobre las operaciones de una empresa de reparto de comida a domicilio.

El objetivo es transformar un conjunto de datos en información útil para facilitar la toma de decisiones mediante indicadores clave (KPIs), tablas dinámicas, gráficos y segmentadores.

---

## 🎯 Objetivos

- Analizar el rendimiento operativo del servicio de entrega.
- Evaluar el comportamiento de los clientes.
- Analizar los ingresos y el ticket medio.
- Estudiar el impacto de los retrasos en la eficiencia del servicio.
- Comparar clientes Premium y No Premium.
- Analizar el efecto del uso de códigos promocionales.

---

## 📂 Dataset

El conjunto de datos contiene **15.000 pedidos** con información relacionada con:

- Clientes
- Pedidos
- Tiempo de entrega
- Ingresos
- Ticket medio
- Rating del cliente
- Retrasos
- Cancelaciones
- Reembolsos
- Clientes Premium
- Uso de códigos promocionales

---

## 🧹 Limpieza y transformación de datos

Durante la preparación de los datos se realizaron las siguientes tareas:

- Revisión de valores nulos.
- Comprobación de formatos.
- Creación de variables auxiliares para el análisis.
- Preparación de tablas dinámicas.
- Creación de indicadores (KPIs).
- Organización del modelo para la construcción del dashboard.

---

## 📈 Análisis realizado

El análisis incluye:

- Evolución de pedidos por mes.
- Distribución de pedidos por hora.
- Comparación entre pedidos con y sin retraso.
- Ticket medio por tipo de cliente.
- Comparativa entre pedidos con y sin promociones.
- Distribución de incidencias.
- Indicadores principales del negocio.

---

## 📊 Dashboard

El dashboard permite analizar de forma interactiva:

- Ingreso total.
- Total de pedidos.
- Ticket medio.
- Tiempo medio de entrega.
- Rating medio del cliente.
- Porcentaje de pedidos retrasados.

Además, incorpora filtros dinámicos para analizar la información según:

- Mes.
- Cliente Premium.
- Uso de códigos promocionales.

---

## 💡 Principales conclusiones

- Los clientes Premium presentan un ticket medio superior.
- Los retrasos aumentan significativamente el tiempo medio de entrega.
- La distribución horaria permite identificar las horas de mayor demanda.
- Las promociones apenas modifican el importe medio del pedido.
- El dashboard facilita una visión global del rendimiento operativo.
